6.1 Vehicle Management APIs

GET/api/vehicles

Get all vehicles for the authenticated driver

Query Parameters:
  • status - Filter by vehicle status
  • search - Search by make, model, or license plate
  • limit - Number of results per page
  • offset - Pagination offset
POST/api/vehicles

Create a new vehicle

Request Body:

PUT/api/vehicles/[id]

Update vehicle information

DELETE/api/vehicles/[id]

Delete a vehicle (soft delete)

6.2 Document Management APIs

GET/api/vehicles/[id]/documents

Get all documents for a specific vehicle

POST/api/vehicles/[id]/documents

Add a new document to a vehicle

Content-Type: multipart/form-data
POST/api/documents/ocr

Extract text from document image using OCR

Features:
  • Automatic field detection (permit numbers, dates, etc.)
  • Confidence scoring for extracted data
  • Support for multiple languages
  • Document type classification

6.3 Notification APIs

GET/api/notifications

Get notifications for the authenticated user

Query Parameters:
  • type - Filter by notification type
  • read - Filter by read status
  • priority - Filter by priority level
PUT/api/notifications/[id]/read

Mark a notification as read
