Project Overview

Purpose

Design and implement comprehensive driver dashboard pages for a cross-border vehicle service platform connecting Hong Kong and Mainland China. The interface follows Tesla-inspired design principles with pink accent colors and focuses on clarity, efficiency, and real-time functionality.

Key Features

  • Real-time earnings tracking and analytics
  • Comprehensive settings management
  • Document upload with drag-and-drop
  • Verification status tracking
  • Mobile-first responsive design

Target Users

Professional Drivers

Licensed drivers operating cross-border transport services

Fleet Operators

Companies managing multiple vehicles and drivers

Independent Contractors

Individual drivers providing premium transport services

Responsive Design Guidelines

Mobile (320px - 768px)

  • ‚Ä¢ Single column layouts
  • ‚Ä¢ Stack cards vertically
  • ‚Ä¢ Full-width buttons
  • ‚Ä¢ Condensed navigation
  • ‚Ä¢ Touch-friendly controls (44px+)
  • ‚Ä¢ Swipe gestures for charts

Key Adjustments

/* Mobile-first approach */ .earnings-grid { grid-template-columns: 1fr; } .chart-container { height: 200px; /* Smaller on mobile */ } .settings-nav { position: sticky; top: 0; }

Tablet (768px - 1024px)

  • ‚Ä¢ Two-column layouts
  • ‚Ä¢ Side-by-side charts
  • ‚Ä¢ Tabbed navigation
  • ‚Ä¢ Expanded card content
  • ‚Ä¢ Larger touch targets
  • ‚Ä¢ Horizontal scrolling tables

Key Adjustments

@media (min-width: 768px) { .earnings-grid { grid-template-columns: 1fr 1fr; } .settings-layout { grid-template-columns: 200px 1fr; } }

Desktop (1024px+)

  • ‚Ä¢ Multi-column layouts
  • ‚Ä¢ Sidebar navigation
  • ‚Ä¢ Hover interactions
  • ‚Ä¢ Expanded data tables
  • ‚Ä¢ Multiple charts visible
  • ‚Ä¢ Modal dialogs for details

Key Adjustments

@media (min-width: 1024px) { .earnings-grid { grid-template-columns: repeat(4, 1fr); } .main-layout { grid-template-columns: 250px 1fr; } .chart-container { height: 400px; } }
