üîò Button System

üìñ Usage Guidelines

Component Hierarchy

Primary Actions: Use pink gradient buttons for main CTAs like "Publish", "Save", "Create New"

Secondary Actions: Use charcoal buttons for alternative actions like "Cancel", "Save Draft"

Tertiary Actions: Use outline or ghost buttons for supporting actions like "Edit", "View", "Delete"

Color Application

Hot Pink (#FF69B4): Primary actions, links, active states, focus indicators

Deep Pink (#FF1493): Hover states, gradients, emphasis elements

Light Pink (#FFB6C1): Background tints, subtle highlights, border accents

Charcoal (#171A20): Primary text, navigation, secondary buttons

Spacing Application

4px (xs): Minimal gaps between closely related elements

8px (sm): Compact layouts, icon spacing

16px (md): Standard spacing between components

24px (lg): Section spacing, card padding

32px (xl): Major section separation

‚ôø Accessibility Standards

All components meet WCAG 2.1 AA standards:

  • Color Contrast: 4.5:1 for normal text, 3:1 for large text
  • Focus Management: Visible focus indicators for all interactive elements
  • Screen Reader Support: Proper ARIA labels and semantic HTML
  • Keyboard Navigation: Full keyboard accessibility with logical tab order
